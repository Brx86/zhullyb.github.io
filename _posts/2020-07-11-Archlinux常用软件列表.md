# Arch系常用软件推荐

#### 注：使用如下命令安装时，请先配置好archlinuxcn源并安装好yay

清华aur的帮助界面打开总是跳转404，不清楚什么情况，我在此记录下yay配置清华源的命令

```
yay --aururl "https//aur/tuna,tsinghua.edu.cn" --save
```

```
sudo pacman -S visual-studio-code-bin
```

```
sudo pacman -S baidunetdisk-bin
```
